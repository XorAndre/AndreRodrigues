DESTAQUES

Angular 7, mas já?

É isso mesmo produção já estão trabalhando na versão 7 do AngularJS? Sim amiguinho! Segundo informações do site Modus, ainda neste ano poderemos ver a versão 7 do Framework. Alguns poderão pensar que é uma apelação para que o Framework volte ao topo dos mais usados já que ultimamente no mundo da web podemos ver o forte uso de React e VueJS. Claro que procurei saber também sobre as novidades que poderemos ver na nova versão, confira abaixo.


4 alterações que irão para a versão do Angular-7


Novo ng-compiler

O novo compilador poderá ser capaz de compilação avançada em tempo rotativo de 8 fases. Isto é tendo uma boa redução nas aplicações chegando a mais ou menos (95-99%), muito interessante, não é?!

Divisão de @angular/core

O pessoal do time Angular marcou um golaço nessa novidade! Nessa eles simplesmente dividiram os núcleos do Angular em 418 módulos. Isso nos traz uma proximidade do DDOS em nossos computadores quando executamos npm -i.

Novo @aiStore

Não irei mentir, mas um dos principais motivos que me levaram a fazer o novo layout do meu site do GitHub, foi pela vontade de compartilhar um pouco sobre essa alteração do Angular. Sim sei que poderia ter publicado no meu Blogger ou até mesmo no perfil do GSTI, mas tinha uma coisa me dizendo faça um novo layout do seu site no GitHub e publique lá primeiro. Bom essa funcionalidade é reservada para IAs (Inteligências artificias), o @aiStore poderá prever como e quando valores poderão ser mudados usando dados vindo dos histórico, api de câmeras e microfones, podendo atualizar as views antes da mudança ou seja mudança antes da mudança.

@angular/mine

Essa é para aqueles que trabalham com mineração de Bitcoin. Este novo pacote irá permitir que os desenvolvedores trabalhem com a mineração em suas aplicações. Isso será executado de forma silenciosa, não sendo rastreado o código core da aplicação. Veja o exemplo: 

                                    import { Mine } from '@angular/mine';

                                    @Component({
                                        name: 'app'
                                        templateUrl: 'app.component.html'
                                    })
                                    
                                    export class AppComponent {
                                        @Mine('your bitcoin address');
                                    }
